# Project Charter

## Objective
Clone and replace the current teaching SaaS in phases, then evolve into an agent-based management system.

## Constraints
1. Existing third-party SaaS cannot be modified.
2. Rebuild core features and data first, then switch team usage.
3. Iterate toward Agent-as-a-Service after stable replacement.
4. All coding is completed by coding agents.

## In-scope (must-have)
- Course management
- Student management
- Order management
- Class attendance records
- Consumption records

## Out-of-scope (for now)
- WeChat Mini Program
- Micro website
- Xiaomai show
- Coupons
- Douyin-related features

## Tech stack
- Backend: Python + FastAPI
- Frontend: TypeScript + Next.js + Tailwind + shadcn/ui
- DB: PostgreSQL (preferred), MySQL optional
